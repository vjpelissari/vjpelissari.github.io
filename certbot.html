Install Certbot Using Snap

1. Ensure Snap is Installed

Ubuntu typically comes with Snap pre-installed. Check if it's available:

snap version

If Snap is not installed, run the following commands:

 
sudo apt update
sudo apt install snapd
sudo snap install core
sudo snap refresh core

2. Remove Any Old Certbot Installations

If you previously attempted to install Certbot via apt, remove it:

sudo apt remove certbot

3. Install Certbot Using Snap

Install Certbot using Snap with the following command:

sudo snap install --classic certbot

4. Create a Symlink for Certbot

Create a symbolic link for easier access:

sudo ln -s /snap/bin/certbot /usr/bin/certbot

5. Run Certbot

For Nginx, run:

sudo certbot --nginx
