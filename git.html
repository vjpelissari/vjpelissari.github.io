Git Common Commands

Create a very new repository

1. Go to github and create a new repository. Add name and description.

2. Create a folder on local machine. Put the initial files for the project there.

3. Open the terminal in the folder and type:


# git init
# git add .
# git commit -m "Initial commit"

Copy the lines below from the GitHub repository page you just created:


# git remote add origin https://github.com/vjpelissari/flixx-app.git
# git branch -M main
# git push -u origin main

Clone


# git clone "git repo url"

Clone the Repository Using the Token

# git clone https://YOUR-TOKEN@github.com/vjpelissari/zoneamentos-api.git

Commits


# git add .
# git commit -m "Commit description"
# git push

Correct Commit Comment

git commit --amend -m "new description"

If you’ve already pushed the commit to a remote repository, you’ll need to force push the change to update it. Be cautious, as this can cause issues for others working on the same branch:

git commit --amend -m "new description"

Fetch


# git fetch origin
# git merge origin/main

Check if the branch is a commit behind


# git fetch origin
# git status
# git log HEAD..origin/main

Check last commit log


# git log -1
