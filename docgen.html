Docgen

Setup

1. Download Docgen at:

https://github.com/thedevsaddam/docgen/releases

2. Create a folder "docgen"

3. Rename the downloaded file to docgen.exe and copy it to the docgen folder

4. Generate the .json collection file in Postman and copy it to the docgen folder

Run

Run the command to generate the index.html API documentation file:

docgen build -i colletion_file_name_created_in_postman.json -o index.html
