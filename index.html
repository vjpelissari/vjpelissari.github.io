NOTEPAD++

    Regular Expression ..... \s+$ ............      ......... delete spaces at the end of the line.
    Regular Expression ..... \s+$|^\s+ .......      ......... delete spaces at the end and beggining of the line.
    Regular Expression ..... (.+) ............ \1,  ......... put comma at the end of line.
    Regular Expression ..... (.+) ............ "\1" ......... suround the line with "".
    Regular Expression ..... (\d)$ ........... \1,  ......... if there is a digit at the end of the line, add a comma. 
    Regular Expression ..... ^\s*$\r?\n ......      ......... delete blank lines.
    Regular Expression ..... ^(.{10})$ ....... 0\1  ......... add left zero to a cpf with 10 digits.
    Extended................ \r\n ............      ......... put all lines together.
    Normal ................. \t .............. \n   ......... replace tab for break line.

GIT

    CREATE A VERY NEW REPOSITORY

        1. Go to github and create a new repository. Add name and description.
        2. Create a folder on local machine. Put the initial files for the project there.
        3. Open the terminal in the folder and type:

        # git init
        # git add .
        # git commit -m "Initial commit"

        Copy the lines below from the GitHub repository page you just created:

        # git remote add origin https://github.com/vjpelissari/flixx-app.git
        # git branch -M main
        # git push -u origin main
    
    CLONE

        # git clone "git repo url"

    COMMITS

        # git add .
        # git commit -m "Commit description"
        # git push

    FETCH

        # git fetch origin
        # git merge origin/main
        
    CHECK IF THE BRANCH IS A COMMIT BEHIND

        # git fetch origin
        # git status
        # git log HEAD..origin/main

    CHECK LAST COMMIT LOG

        # git log -1
